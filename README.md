# computer_shop
ОПИСАНИЕ:
"Автоматизированная система для магазина ПК"
Проект представляет из себя оконное приложение с функционалом ведения отчетности для персонала магазина и возможностью получить информацию о наличии товара для клиентов.
(В качестве демонстрации приложен файл "КР по БД.docx")

СТЕК ТЕХНОЛОГИЙ:
Язык: Java
Фреймворк: Swing
Базы данных: MySQL

СТАТУС:
Проект не закончен.
Выполнено в качестве курсовой работы по дисциплине "Базы данных".
